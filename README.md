# YouTube Sentiment Analysis 
## Features
- Sentiment analysis (Positive, Neutral, Negative)
- Multi-language detection
- Batch processing for up to 1000 comments

## Usage
1. Enter YouTube video URL
2. Click "Analyze"
3. Wait 30-60 seconds for results
